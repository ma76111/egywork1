# 🇪🇬 إيجي وورك - منصة المهام المصرية

## تشغيل المشروع

### 1. تشغيل الـ Backend
```bash
cd backend
node seed.js        # إنشاء حساب الأدمن (مرة واحدة فقط)
node index.js       # تشغيل الخادم على المنفذ 5000
```

### 2. تشغيل الـ Frontend
```bash
cd frontend
npm run dev         # تشغيل على المنفذ 5173
```

## حساب الأدمن

- كلمة المرور: `admin123`
- ⚠️ غير كلمة المرور فور الدخول
